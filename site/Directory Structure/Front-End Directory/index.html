    Description

    API

    apiChứa các tệp TypeScript xử lý giao tiếp với API backend

    Assets

    assetsChứa các tệp tĩnh như hình ảnh, phông chữ, tệp SVG, CSS, SCSS và icon.

    CI

    ciChứa các tệp liên quan đến tích hợp liên tục (CI) và triển khai liên tục (CD), bao gồm Dockerfile, Chart Helm, và version.txt.

    Components

    componentsChứa các thành phần Vue có thể tái sử dụng

    Composables

    composablesChứa các hàm có thể sử dụng lại trong các thành phần

    Constants

    constantsChứa các hằng số dùng chung trong ứng dụng, bao gồm API constants

    Layouts

    layoutsChứa các layout chung cho các trang khác nhau, hiện có layout mặc định (default.vue).

    Libs

    libsChứa các thư viện riêng của dự án

    Locales

    localesChứa các tệp bản dịch(Anh,Nhật,Việt)

    Middleware

    middlewareChứa các middleware của Nuxt, hiện có middleware xác thực toàn cục (auth.global.ts).

    Pages

    pagesChứa các tệp định nghĩa các trang của ứng dụng

    Plugins

    pluginsChứa các plugin Nuxt để mở rộng chức năng của ứng dụng

    Public

    publicChứa các tệp tĩnh sẽ được sao chép trực tiếp vào thư mục public khi ứng dụng được xây dựng

    Utils

    utilsChứa các hàm tiện ích chung, bao gồm các hàm common và hàm kiểm tra dữ liệu (validators.ts).
